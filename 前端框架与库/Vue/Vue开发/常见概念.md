#### 9.事件监听
v-on:click
@click
修饰符
.stop 阻止冒泡
```js
<div @click="divClick">
    aaa
    <button @click.stop="butClick">点我</button>
</div>  //点击按钮阻止divClick方法执行
```
.prevent 阻止默认行为
```js
<form action="baidu">
    <input type="submit" value="12" @click.prevent="">
</form>  //阻止点击默认提交手动提交
```
keyup监听某个键盘的键帽
```js
<input type="text" @keyup.enter="keyup">   //监听回车键
```
