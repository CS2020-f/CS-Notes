### Element-UI
一套基于vue2.0 的桌面端组件库，使用前端框架封装的代码帮助工程师快速开发

提供6大类组件，基础组件，表达类组件，数据类组件，提示类组件，导航类和其他类组件

```html
//按钮
  <el-button>默认按钮</el-button>
  <el-button type="primary">主要按钮</el-button>

//日期
 <el-date-picker
      v-model="value1"
      type="date"
      placeholder="选择日期">
    </el-date-picker>

```
